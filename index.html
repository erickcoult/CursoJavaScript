Janelas

No JavaScript podemos criar aquelas janelas de confirmação que nos permitem fazer perguntas ou mostrar um aviso. Sempre quando vamos fazer algum código no JavaScript dentro do HTML, usamos a tag "script". Para fazermos as tags de janelas, utizaremos;
  • window.alert('') => para criar um alerta, aparecendo apenas"ok"
  • window.confirm('') => para criar uma confirmação, aparencendo "ok" e "cancelar"
  • window.prompt('') => para criar uma janela de questionário ou interação com usuário

Uma coisa que temos que ter o costume é de trocar o significado de = (igual) para RECEBE. Ou seja, lemos "var n1 = 4" como "var n1 recebe 4"

Variáveis

Uma variável é um espaço na memória do computador destinado a um dadoo seu nome que é alterado durante a execução do algoritmo. Para funcionar corretamente, as variáveis precisam ser definidas por nomes e tipos

Entretanto, há uma regra para se dar um identificar para a variável, sendo elas:

  • Podem começar com letras, $ ou _
  • Não podem começar com números
  • É possível usar letras ou números
  • É possível usar acentos e símbolos
  • Não podem conter espaços
  • Não podem ser palavras reservadas, ou seja, palavras que seriam códigos para o JavaScript
  • Inteiros: números inteiros (0, 1, 2 , 3…)
  • Reais: números inteiros e decimais (0; 1,2; 2,1; 3)
  • Caracteres: números reais, letras e outros símbolos
  • Lógico: comandos de VERDADEIRO ou FALSO

Um exemplo para a criação de um var seria: var exemplo = 14. Assim, quando colocarmos "exemplo" no terminal, ele nos dará "14" como resposta

Caso precisemos utilizar um terminal para executar os comandos, pressionamos "ctrl + shift + ' "

IMPORTANTE: Por estarmos utilizando JS, temos que inserir "node" no terminal. Isso se diferencia de linguagem para linguagem

Tratamento de dados

Tratamento de dados nada mais é do que fazer com que o JS processe as informações dadas nas prompt's (janelas de interação). Ou seja, fazer uma soma, divisão, conversão e etc.

Para isso, precisamos criar uma variável, dar a identificação e então, a pergunta. Devemos ter a quantidade de janelas para a quantidade de números que vamos somar. Ou seja, se eu irei somar 2 números, vou ter que criar 2 janelas de prompt.

Além de criar uma Variável para fazer a soma, preciamos criar uma Terceira variável que teria como a finalidade de fazer a soma das outras duas. Também precisamos criar um alerta para nos mostrar o valor.

Assim, o resultado ficaria:

  • var n1 = Number.parseFloat(window.prompt ('número 1'))
  • var n2 = Number.parseFloat(window.prompt ('número 2'))
  • var s = n1 + n2
  • window.alert('a soma dos valores é ' + s)

Podemos perceber que no código há uma "parseFloat". Ela tem como função de fazer que o JS apenas somará números quebrados. Há também outros dois, sendo eles " parseInt" e o próprio "Number". Assim, temos:

  • parseFloat: trabalhará apenas com valores quebrados, que possuam vírgula.
  • parseInt: trabalhará apenas com valores inteiros.
  • Number: O próprio JS decidirá qual irá utilizar.

Concatenação

A Concatenção tem como função de juntar string em uma frase. Ela pode ser usada quantas vezes precisar, necessitando apenas de acrescentar "+" entre cada frase. Segue o Exemplo para montar a frase "olá! tudo bem ?"

var p1 = olá! var p2 = tudo bem ? var final = p1 + p2 final = olá!tudo bem?

Logo, podemos criar frases longas, bastando apenas acrescentar áspas("") e entre cada frase, adicionar o "+"

Tamplate String

O método anterior pode acabar por ser algo muito demorado caso precisemos montar uma frase grande. Para isso, usamos os tamplate string. Eles permitem fazer com que ja montemos a frase com ás variáveis dentro, bastando apenas acrescentar ${...} e é importante acrescentar a frase entre CRASE(``). Ficando desse jeito:

`o ${nome} que possui ${idade} gosta muito de comer ${fruta}` Resultado: o João que possui 14 anos gosta muito de comer banana

Formatando Strings e Números

Podemos acrescentar características as Strings como por exemplo as:

  • x.lenght =>quantos caracteres a strings
  • x.toUpperCase() => tudo para MAIÚSCULA
  • x.toLowerCase() => tudo para MINÚSCULAS

Agora para a formatação de números, possuimos os seguintes códigos

  • x.toFixed(y) => y representa a quantidade de casas que aparecerá após vírgula
  • x.toFixed(y).replace('.', ',') => converte o ponto em vírgula.
  • x.toLocaleString('pt-BR', {style: 'currency', currency: 'BRL'}) => acrescenta o símbolo monetário desejado(usd, eur, brl, etc)

    Aritméticos

    Os operadores nada mais são do que a matemática pura que aprendemos na escola. Nela possuimos os sinais:

    • + => adição
    • - => subtração
    • * => multiplicação
    • / => divisão
    • % => divisão inteira
    • ** => elevado ao quadrado

    Precedência

    Assim como na matemática, temos uma ordem de se "realizar" as contas. Sendo assim, temos do mais "forte" ao mais "fraco" dentro de uma expressão, sendo elas:

    • Aritméticos: (),**,*,/,%,+,-
    • Relacionais: >, <, >=, <=, ==, !=, ===
    • Lógicos: !, &&, ||

    Para fixar, veja o exemplo até agora::

    (5+3)*2/4 (8)*2/4 16/4 4

    Atribuição simples

    Outra coisa que é semelhante a matemática é a Atribuição simples. Ela nada mais é do que atribuir um valor à uma variável. Ou seja, uma função. Veja os exemplos:

    var a = 2 + 2 => valor de a é 4 var b = a + 6 => valor de b é 10 var c = a + b % 3 => valor de c é 1

    Auto-atribuição

    a Auto-atribuição tem como objetivo de dar uma "nova" atribuição à variável. Logo, se você deu um valor à variável e, logo em seguida, dá uma nova atruibuição, aquela ultima substituirá a primeira. Veja abaixo:

    var n = 3 => o "n" vale 3. n = n + 4 => o "n", que antes valia 3, agora passa a valer 7. n = n - 5 => o "n", que antes valia 7, agora passa a valer 2. Também podemos simplificar var n = 3 n += 4 n -= 5

    Isso pode ser utilizado com qualquer operador que vimos acima

    Relacionais

    Relacionais são operadores da grande maioria das linguagens das programções, sendo mostrado por símbolos matemáticos

    • > ---> maior que
    • < ---> menor que
    • >= ---> maior ou igual que
    • <= ---> menor ou igual que
    • == --->igual à
    • != ---> não igual/diferente
    • === ---> igualdade restrita, ou seja,tendo o mesmo valor e do mesmo tipo (typeof)

    Ao utilizarmos um Operador Relacional, nosso resultado será sempre um valor booleano, ou seja, verdadeiro(true) ou falso(false)

    5 > 2 => true 8 <= 5 =>false 4 != 4 => false 7 == 7 => true 2 === '2' => false "2" === "2" => true

    Operadores lógicos

    No JavaScript possuimos 3 Operadores lógicos, sendo eles

    • ! => Negação
    • && => Conjunção
    • || => Disjunção

    O operador de Negação é tratado como Operador UNÁRIO, ou seja, após inserimos !, resultado será ou false ou true

    Já o operador de Conjunção é considerado como BINÁRIO, assim, possuindo doisoperadores lógicos( um de cada lado). Um exemplo é quando você pede uma caneta vermelha euma azul, ou seja, você só ficará satisfeito se tiver uma caneta azul e uma vermelha

    Os operadores de Disjunção também são considerados BINÁRIO que possuem dois operadores lógicos. Seguindo o exemplo de canetas, as Disjunção busca ou vermelha ou azul. Assim, tendo uma das duas, você ficará satisfeito. Logo, basta apenas um true para o resultado ser verdadeiro

    Uma coisa que devemos saber é que os operadores lógicos também possuem uma order de grandeza, sendo primeiro o NÃO (Negação), depois o E (Conjunção) e po ultimo o OU(Disjunção)

    IMPORTANTE: Quando se tem Operadores Relacionais, Operadores Lógicos e Operadores Aritméticos na mesma expressão, é seguido primeiro por ARITMÉTICO, RELACIONAIS E LÓGICOS.

    var a = 5 var b = 8 a > b && b % 2 == 0 a > b && 8 % 2 == 0 O lado direito é verdadeiro já que o resto de 8 % 2 é igual a 0 5 > 8 && true false && true Logo, o resultado final será falso já que os dois tinham que ser true

    Operador Ternário

    Constituído apenas por dois símbolos, sendo o ? e :. Ele possui este nome por conter 3 partes sendo

    teste ? true : false

    Com isso, podemos fazer uma expresão que se der um determinado valor (verdadeiro ou falso) no bloco de teste, aparecerá o valor inserido no bloco true:false. Veja abaixo

    var x = 4 var res = x / 2 == 2 ? 5:8 Nesta situação, caso o resultado de x/2 for 2, então o valor de res será 5(true), senão, mostrará 8(false) var res = 4/2 == 2 ? Assim, o valor de res é 5, já que a expressão do bloco teste é verdadeira res = 5

    Outro exemplo muito usado seria fazer o Operador Ternário como um "seletor de dados"

    var idade = 19 var res = idade >= 18 ? 'MAIOR':'MENOR' res = 'MAIOR'

    DOM

    O Document Object Model tem como função de juntar os métodos acima com o nosso site. Ou seja, é um conjunto de objetos dentro do seu navegador que dará acesso aos componentes internos do seu website.

    Árvore DOM

    Tudo dentro do JS está dentro de um objeto window. Dentro dele, possuimos vários objetos, como por exemplo, Location, documento (Onde há todo o HTML), history e etc. Assim, por meio da tag window, podemos acessar diversar informações no site e também expor, como "windwos.document.write(window.document.URL)" assim, dissemos ao JS para fazer um write mostrando a URL do site.

    Selecionando

    Podemos selecionar para navegar dentro da Árvore DOM, havendo diversos métodos para isso, como:

    • por Marca
    • por ID
    • por Nome
    • por Class
    • por Seletor

    Marca

    Ela é representada pelo código getElementsByTagName(). Sua função é de selecionar mais de um objeto. Com isso, é possível fazermos alterações nas tag selecionadas, podendo mudar a cor, tamanho, fonte e etc. Dentro dos parenteses inserimos a tag que desejamos fazer alguma mudança. Já dentro dos colchetes selecionamos o "número" dela, ou seja, se o 'p' que desejamo mudar for o 6º do documento, então iremos inserir 12, pois a contagem começa a partir de 0.

    var t1 = document.getElementsByTagName('p')[1] Para realizar alguma mudança basta inseir a var e a mudança t1.style.color = 'blue' Assim como podemos escrever algo dentro de uma tag document.write(t1.innerText) Isso também pode ser usado junto com a concatenação document.write('Este é um exemplo' + t1.innerText)

    As demais formas possuem quase a mesma ideia, porém as que possuem plural como "elemenTS", será acompanhada de colchetes após os parenteses.

    Eventos DOM

    Evento é tudo aquilo que pode acontecer com qualquer elemento (div, span, code etc). Algo mais comum são os eventos de mouse, ou seja, quando passamos o mouse no elemento, irá ser feito diversos disparos, como os

    • mouseenter: quando mouse entra dentro do elemnto
    • mousemove: quando o mouse se move dentro do elemento
    • mousedown: quando ficamos com o botão do mouse pressionado no elemento
    • mouseup: quando soltamos o botão do mouse
    • click: evento em que se preciona e solta o botão do mouse normalmente
    • mouseout: quado tiramos o mouse do elemento

    Estes são apenas alguns dos diversos eventos possíveis no JS. Para vermos todos os eventos podemos acessar o site Event Reference do Mozilla( clique aqui)
