Janelas

No JavaScript podemos criar aquelas janelas de confirmação que nos permitem fazer perguntas ou mostrar um aviso. Sempre quando vamos fazer algum código no JavaScript dentro do HTML, usamos a tag "script". Para fazermos as tags de janelas, utiizaremos;
  • window.alert('') => para criar um alerta, aparecendo apenas"ok"
  • window.confirm('') => para criar uma confirmação, aparencendo "ok" e "cancelar"
  • window.prompt('') => para criar uma janela de questionário ou interação com usuário
